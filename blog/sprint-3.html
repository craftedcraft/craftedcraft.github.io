An analogy to describe the differences between HTML and CSS

A simple analogy to describe the differences between HTML and CSS would be to think of them being the components to build a house. HTML would be the building blocks; foundations, pillars and overall structural integrity of the building. While CSS would then be the aesthetics of the building like colour, textures and design.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Explaining control flow and loops using a process from everyday life could be related to ‘brushing your teeth’. The control flow is used to describe the order in which your code executes so just like brushing your teeth you need to go through a step by step process in a particular order.

eg. The process in the control flow of brushing your teeth would be:

  • Run hot water over your toothbrush bristles until soft
  • Apply toothpaste to your tooth brush bristles
  • Brush your teeth for two minutes until clean
  • Rinse your tooth brush clean
  • Repeat morning and night
